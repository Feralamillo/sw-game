# Star Wars Game

This project has been developed using [Create React App](https://github.com/facebook/create-react-app) with typescript.

![Preview][preview.svg]

## Launch

To start the project:

### Development

Run `npm run start`.<br>
A new tab will open at [http://localhost:3000](http://localhost:3000) in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.

### Testing

Run `npm run test` to launch the test suite in watch mode.<br>
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.

### Build

Run `npm run build` to build the app for production, the files will be generated in the `build` folder.<br>
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.<br>
The app will be ready to be deployed!

See the section about [deployment](https://facebook.github.io/create-react-app/docs/deployment) for more information.
